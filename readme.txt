------------------------- DEPENDENCIES ----------------------------


Java JDK 18 or above (not tested on version lower than 18)


-------------------------------------------------------------------------------


----------------------- HOW TO COMPILE ----------------------------
1. Ensure Java is installed onto environment:
Check here:
https://www.java.com/en/download/verifyjavaversion.jsp


2. Download all files
3. Open command line (i.e cmd on Windows)
4. Navigate to ~\MaxwellBase\src directory
5. Compile the project using the command below:
~\MaxwellBase\> javac src/MaxwellBase/* src/Constants/* -d out/
6. Finally, run the program using the command:
~\MaxwellBase\src\MaxwellBase> java MaxwellBase.java
7. For more instruction, instruct the user guide.
